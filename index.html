Real-time location

Real-time location of the position of your family and loved ones Thanks to Family Locator, your family members can share their location securely, using the one object we all have with us - the smartphone. It allows you to stay in touch with your loved ones and view their location in real time.

Features:

Sign up form for new user to create an account.
A new user record is created in Google Firebase.
If an already existing user tries to sign up, he is prevented in doing the same.
Login Page to allow only authorized users to login.
Performs validation for username and password match.
Home Page where users can check current location and check current location in map. Map Activity shows current location on Google maps..

How it works?

The first device get continous location update from gps, network etc.
then Write and update the location data to a specific firebase firestore document
Last, the second device read that firebase firestore document and show the marker in that location.
